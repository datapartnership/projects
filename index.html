Projects

The World Bank Data Lab team, which supports the Development Data Partnership, collaborates with teams to solve global challenges using best practices in coding, code documentation, and data visualization.

Unlike traditional data analysis, which results in a single-use report or visualization, Partnership products are designed to be customized, reused, and updated, thereby building the capacity of partner organizations to quickly deliver complex data science solutions to pressing global challenges.

To learn more, following are sample projects. Note some projects are not public and require permissions to access. Got questions? Write to the Development Data Partnership.
